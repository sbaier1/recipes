## Zutaten

- <span class="qty" data-base="50">50</span> g Wasser
- <span class="qty" data-base="8">8</span> g Öl (z.B. Raps)
- <span class="qty" data-base="5">5</span> g Protein
- <span class="qty" data-base="1">EL</span> Sonnenblumenkerne
- <span class="qty" data-base="1">Prise</span> Paprika o.ä.
- <span class="qty" data-base="0.5">TL</span> Glutamat
- <span class="qty" data-base="1">TL</span> Salz
- <span class="qty" data-base="0.5">TL</span> Agar Agar
- <span class="qty" data-base="0.125">1/8</span> TL Johannisbrotkernmehl
- <span class="qty" data-base="0.125">1/8</span> TL Guarkernmehl
- <span class="qty" data-base="0.25">1/4</span> TL Bambusfasern
- <span class="qty" data-base="1">TL</span> Konjakpulver
- <span class="qty" data-base="1">TL</span> Knoblauchpulver
- <span class="qty" data-base="1">TL</span> Zwiebelpulver
- <span class="qty" data-base="1">TL</span> Paprikapulver (edelsüß oder geräuchert)
- <span class="qty" data-base="1">Prise</span> Oregano, Majoran, Fenchel, Kubebenpfeffer, Pfeffer
- <span class="qty" data-base="0.25">1/4</span> TL Natriumacetat
- <span class="qty" data-base="1">Prise</span> Zucker
- <span class="qty" data-base="1">Schuss</span> Sojasauce (optional)

## Zubereitung

- [ ] Sonnenblumenkerne eventuell vorher einweichen.
- [ ] Wasser abwiegen und zum Kochen bringen.
- [ ] Die trockenen Zutaten vermischen.
- [ ] Öl und Füllung (Schnittlauch, Paprika, etc.) vorbereiten.
- [ ] Wasser, Öl-Mischung und trockene Zutaten im Mixer vermengen.
- [ ] Notizen zur Iteration 4/5: Zu viele Sonnenblumenkerne. Bei Iteration 4 optimal, aber zu viel Agar Agar. Gekochtes Wasser in Mixer geben.
- [ ] Notizen zur Iteration 2: Mehr Verdickungsmittel (2-3 TL Guarkernmehl, 2 TL Johannisbrotkernmehl, mehr Agar Agar, Bambusfasern) und weniger Salz (1/2 TL) verwenden.
- [ ] Notizen zur Iteration 3: Mehr Wasser verwenden. Salz passt, weniger Gewürze. Umrechnung von Rügenwalder Aufschnitt: 24,5g Nährstoffe -> 75,5% "Hydration"; 7,3% Ballaststoffe -> 6,57g Verdickungsmittel gesamt; Reihenfolge nach Menge: Agar, Konjak, Guarkern, Bambus; 2% Salz, 2% Eiweiß, 9% Fett.
